# Credit Risk and Customer Behaviour Analysis

## Project Overview

This project analyzes 50,000 credit card customers to identify risk patterns, customer behaviour, and financial insights using Python.

## Tools Used

- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn

## Dataset

Contains 50,000 customers with:

- Income
- Credit Limit
- Balance
- Late Payments
- Risk Level



## Key Insights

- High utilization customers are high risk
- Late payments strongly indicate risk
- Income affects credit limit

## Project Files

- Credit_Risk_EDA.ipynb → Python analysis
- finance_eda_50000.csv → Dataset
